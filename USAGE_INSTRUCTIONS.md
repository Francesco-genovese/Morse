# HOW TO INSTALL AND USE 
Here are the steps to run the Python code:

1. **Save the Python Code:**
   - Copy the provided Python code.
   - Paste it into a text file and save it with a ".py" extension, for example, "morse_code_converter.py".

2. **Open Terminal or Command Prompt:**
   - Open the terminal or command prompt on your operating system. You can do this by searching for "Command Prompt" or "Terminal" in the start menu or application bar.

3. **Navigate to the Working Directory:**
   - Use the `cd` command to navigate to the directory where you saved the Python file. For example:
     ```bash
     cd path/to/your/directory
     ```

4. **Run the Program:**
   - Type the following command to execute your Python script:
     ```bash
     python morse_code_converter.py
     ```
     If you are using Python 3, the command might be `python3` instead of `python`.

5. **Follow the Instructions:**
   - The program will start running and prompt you to choose whether to encrypt or decrypt. Follow the on-screen instructions, input the necessary text, and the program will return the result.

Note that the exact procedure might vary slightly depending on the operating system you are using. Make sure to follow the appropriate instructions for your environment.
